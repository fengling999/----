# 明心觉察

一个中文网站，主题为“佛学智慧 × 现代心理学的自我觉察与情绪修行平台”。

本站提供情绪急救、今日觉察、正念练习、修心文章和关于本站说明，帮助用户在日常生活中温和地理解情绪、念头、执着与可行动的一步。

## 产品定位

明心觉察是一个自我觉察与情绪教育网站，不是医疗产品，也不是心理咨询服务。

内容结合：

- 佛学中的无常、缘起、执着、慈悲、正念等观照方式
- 现代心理学中的情绪命名、压力反应、认知理解、自我接纳和行为练习

整体目标是提供安静、克制、可执行的日常练习空间。

## 功能模块

- 首页：平台介绍、核心入口、修心路径、精选文章和安全声明
- 今日觉察 `/daily`：三问式日记，记录保存在当前浏览器
- 情绪急救 `/emergency`：焦虑、愤怒、难过、迷茫、放不下、自责的短时稳定引导
- 正念练习 `/practice`：呼吸觉察、身体扫描、放下执着、慈悲观、睡前安定练习
- 修心文章 `/articles`：支持分类筛选的原创文章系统
- 关于本站 `/about`：定位、适用场景、内容边界、隐私说明和免责声明

## 技术栈

- Next.js App Router
- TypeScript
- Tailwind CSS
- React
- 本地静态 TypeScript 数据
- `localStorage` 保存今日觉察记录

## 本地运行

```bash
npm install
npm run dev
```

打开：

```text
http://localhost:3000
```

## 构建部署

```bash
npm run lint
npm run build
```

项目适合部署到 Vercel。详见：

```text
DEPLOYMENT.md
```

当前项目不需要必填环境变量。

可选：

```text
NEXT_PUBLIC_SITE_URL=https://your-domain.com
```

用于让 `sitemap.xml` 和 `robots.txt` 输出正式域名。

## 内容安全边界

本站必须保持以下边界：

- 不提供心理诊断
- 不替代心理咨询、心理治疗、精神科或其他医疗服务
- 不承诺治愈、疗效或一定改善
- 不使用“你有某某症”这类判断
- 涉及自伤、自杀、伤害他人或紧急危险时，只提示联系当地紧急服务、可信任的人或专业机构

## 隐私说明

- 当前项目没有登录、数据库、支付、社区或 AI 聊天
- 今日觉察记录仅保存在用户当前浏览器的 `localStorage`
- 今日觉察记录不会上传服务器
- 用户清理浏览器数据后，本地记录可能被删除

## 后续迭代方向

优先考虑低复杂度、低风险优化：

1. 增加更多本地文章和练习内容
2. 为今日觉察增加本地导出功能
3. 增加更细致的移动端可访问性检查
4. 为文章添加相关推荐
5. 增加轻量的主题色配置

不建议过早加入登录、数据库、付费、社区或 AI 聊天。
